## How to do this efficiently

Budget 30-45 minutes. Open a new tab for Market Kurly. Search each Korean name. Right-click the main product photo, Save As, rename. Batch the whole list before moving on.

If Kurly doesn't have a clean shot for something, try Coupang. If neither works, grab from Google Images but filter by large size. Worst case for 2-3 items, skip and let Claude Code use a placeholder — we lose nothing.

For `hero.jpg`, spend a little longer. It's the first thing Yoshimura-san sees. Good search terms: `한상차림 오버헤드`, `korean table overhead shot`, `korean food spread flatlay`. Unsplash and Pexels have high-resolution free versions if the Korean sites don't give you something striking.
